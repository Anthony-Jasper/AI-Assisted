Google UX Course

In October 2022, I came across a Coursera course that was taught by Google on everything about becoming a UX Designer, ranging from research to launch. I thought this would be a great opporunity to start learning about UX design and UX in general to help aid future projects. To start progressing through the course, you first have to choose a case study. Using a randomiser, I decided my case study would be on an order tracking app for a BBQ restaurant.

Minecraft Idle Game

Minecraft Idle is a web based game that will require very little interaction from the user. The primary feature of an idle game is its strategy and how the user is able to maximize and plan ahead when leaving the game to run by itself.

The game is currently still a WIP but if you wish to take a look at how the development of it is progressing, visit idle.anthonyjasper.co.uk

Steam Library

I've used Steam almost everyday for the last 8 years. Since using it for so long, I thought it would be a good idea to put my skills to the test and redesign one of the main pages for it. The library page is where you access your list of games, statistics and updates. My main aim was to try and use whats trending at the moment in web design and implement it into the new page design.

Reddit Frontpage

I use Reddit multiple times a day, to view gaming forums, worldnews and cat videos. Reddit has 1.6 billion unique viewers a day but there is mixed feelings on the newest design they implemented a few years ago. A large sum of people still use the old version of the design, including myself because the newest design didnt keep some of the features that most people enjoyed having and found convenient for them. I decided that I would do a redesign for the reddit frontpage and combine the old and the new together.
