Colours

When doing a Christmas themed design, it's almost set in stone to use blue, white, green, yellow and red colours. Following the 60-30-10 rule, I went with blue, white and green. The green is used for button elements as well as the images to create a festive feeling when using the app. The green also allows it to have contrast against the white and blue colours.

Typography

A generic and bland typography was used, with not much thought given to it. If I was to revisit it, I would choose a more festive typography for all of the font on the page. Design Patterns

Input fields. The app requires the user to input a lot of information about themselves as well as other people within their group. A mistake was done here and that was adding labels to the input fields. Due to the sheer amount of input fields required within the app and space, the labels take up too much valuable space which could have been achieved by just using the input fields with placeholder text. It also slows down the users scanning of the app and not drawing the attention to what I am trying to get the user to do.
